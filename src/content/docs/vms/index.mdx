Welcome to the Medjed AI VMs documentation. This section provides comprehensive information about our virtual machine offerings, including pricing, configuration options, and connection methods.

## What are VMs?

Virtual Machines (VMs) are virtualized compute resources that allow you to run applications and workloads in a flexible, scalable environment. Medjed AI offers a variety of VM types to suit different use cases, from general-purpose computing to high-performance GPU workloads.

## Available VM Types

Medjed AI provides several VM categories:

- **CPU VMs**: General-purpose VMs optimized for everyday computing tasks
- **GPU VMs**: High-performance VMs with NVIDIA GPUs for AI/ML workloads
- **Bare Metal VMs**: Dedicated physical servers for maximum performance and control

## Key Features

- **Flexible Configuration**: Choose from various CPU, GPU, memory, and storage options
- **Fast Provisioning**: Deploy VMs in minutes
- **High Performance**: Optimized for AI/ML, data processing, and scientific computing
- **Secure**: Isolated environments with built-in security features
- **Cost-Effective**: Pay-as-you-go pricing with no long-term commitments

## Getting Started with VMs

1. **Choose a VM Type**: Select the appropriate VM category based on your workload requirements
2. **Configure Your VM**: Customize CPU, GPU, memory, and storage options
3. **Deploy Your VM**: Provision the VM through the Medjed AI dashboard
4. **Connect to Your VM**: Use SSH or other connection methods to access your VM
5. **Manage Your VM**: Monitor performance, adjust resources, and manage your VM lifecycle
